Conditionals

Conditional 2 / Second Conditional

definition: el condicional 2 se usa para situaciones hipoteticas para ideas del presente.

if I were rich, I would buy a big house. si yo fuera/estuviera compraría

if I were you, I would do it. (spanish = fuera tu, haría )

if she played video games, she would play well. (spanish = ella jugara )

if you studied english, you would be very good at english.

if they wanted to go, they would go.

if we danced all night, we would be tired.
